# SAS-DOS

SAS-DOS — это ностальгический симулятор операционной системы, написанный на Python. Проект имитирует работу ретро-ОС с собственной файловой системой, защитой от несанкционированного доступа и встроенными утилитами.

## 🖥 Особенности системы
- **Виртуальная файловая система:** SAS-DOS создает изолированную среду для хранения ваших данных.
- **Система лицензирования:** Требуется валидный ключ для активации (используйте `keygen.py`).
- **Инструментарий:** Включает текстовый редактор, сетевой браузер и системные команды.
- **Хардкорный режим:** Система имитирует критические ошибки (BSOD) при повреждении системных файлов или попытке взлома.
